# ☕ حاسبة وصفات القهوة — Coffee Recipe Calculator

موقع تفاعلي لحساب مقادير تحضير القهوة بناءً على عدد الأكواب، طريقة التحضير، ونوع التقديم (حار/بارد).

## 🎯 المميزات

✅ **6 طرق تحضير مختلفة:** V60, Chemex, French Press, Espresso, AeroPress, Cold Brew  
✅ **تقديم حار وبارد:** خطوات مختلفة لكل نوع مع تعديل كميات الثلج والماء  
✅ **حساب تلقائي فوري:** يتحدث فور تغيير أي خيار  
✅ **خطوات تفصيلية:** مع التوقيت الزمني لكل خطوة  
✅ **بيانات قابلة للتعديل:** عبر ملف Excel بسيط  
✅ **تصميم احترافي:** واجهة داكنة أنيقة مع دعم اللغة العربية  
✅ **متجاوب:** يعمل على جميع الأجهزة (موبايل، تابلت، سطح المكتب)

## ✏️ كيفية تعديل الوصفات

### الملف الأساسي: `coffee_recipes.xlsx`

الملف يحتوي على **ورقتين:**

#### **ورقة 1: Methods** (طرق التحضير)

| العمود | الوصف | مثال |
|--------|-------|------|
| **ID** | معرّف فريد للطريقة | `v60` |
| **Name_Ar** | اسم الطريقة بالعربية | `V60` |
| **Name_En** | اسم الطريقة بالإنجليزية | `V60 Pour Over` |
| **Ratio** | نسبة الاستخلاص (1:X) | `15` (يعني 1:15) |
| **Time_Min** | الحد الأدنى لوقت التحضير (دقائق) | `3` |
| **Time_Max** | الحد الأقصى لوقت التحضير (دقائق) | `4` |
| **Grind_Size** | درجة الطحن | `ناعم-متوسط` |
| **Temp** | درجة حرارة الماء (مئوية) | `93` |
| **Supports_Iced** | هل تدعم القهوة الباردة؟ | `Yes` أو `No` |

**مثال:**
```
ID          | Name_Ar | Name_En      | Ratio | Time_Min | Time_Max | Grind_Size   | Temp | Supports_Iced
v60         | V60     | V60 Pour Over| 15    | 3        | 4        | ناعم-متوسط   | 93   | Yes
frenchPress | French Press | French Press | 14 | 4 | 5 | خشن | 95 | Yes
```

#### **ورقة 2: Steps** (خطوات التحضير)

| العمود | الوصف | مثال |
|--------|-------|------|
| **Method_ID** | معرّف الطريقة (يجب أن يطابق ID من ورقة Methods) | `v60` |
| **Type** | نوع التقديم | `hot` أو `iced` |
| **Step_No** | رقم الخطوة | `1`, `2`, `3`, ... |
| **Time** | التوقيت | `0:00`, `0:45`, `1:30` |
| **Action** | وصف الخطوة (يمكن استخدام متغيرات) | `صب {coffee*2} جرام ماء` |
| **Detail** | تفاصيل إضافية | `تأكد من تبليل جميع البن` |

**المتغيرات المدعومة في حقل Action:**
- `{coffee}` → كمية البن بالجرام
- `{water}` → كمية الماء بالجرام
- `{coffee*2}` → ضعف كمية البن
- `{water*0.5}` → نصف كمية الماء
- `{water*0.75}` → 75% من كمية الماء

**مثال:**
```
Method_ID   | Type | Step_No | Time  | Action                                    | Detail
v60         | hot  | 1       | 0:00  | ابدأ بـ Bloom: صب {coffee*2} جرام ماء    | تأكد من تبليل البن
v60         | hot  | 2       | 0:45  | صب الماء حتى {water*0.5} جرام            | صب بحركة دائرية
v60         | iced | 1       | 0:00  | ضع الثلج في الإبريق                      | الثلج سيبرّد القهوة
```

## 🛠️ التخصيص المتقدم

### إضافة طريقة تحضير جديدة:

1. افتح `coffee_recipes.xlsx`
2. في ورقة **Methods**، أضف صف جديد:
   ```
   ID: aeropress
   Name_Ar: AeroPress
   Name_En: AeroPress
   Ratio: 13
   Time_Min: 1.5
   Time_Max: 2
   Grind_Size: ناعم-متوسط
   Temp: 85
   Supports_Iced: Yes
   ```
3. في ورقة **Steps**، أضف خطوات جديدة مع `Method_ID: aeropress`
4. احفظ وارفع إلى GitHub

### تغيير حجم الكوب الافتراضي:

افتح `app.js` وغيّر السطر:
```javascript
defaultCupSizeMl: 240,  // غيّر 240 إلى الحجم المطلوب (بالميلليتر)
```

### تغيير الألوان:

افتح `style.css` وعدّل متغيرات الألوان في قسم `:root`:
```css
--clr-coffee: #c8813a;      /* لون البن */
--clr-ice: #7ecfea;         /* لون الثلج */
--clr-text: #f0e6d3;        /* لون النص */
```

## 🐛 استكشاف الأخطاء

### الموقع لا يعرض البيانات:
1. تأكد من وجود ملف `coffee_recipes.xlsx` في المجلد الصحيح
2. افتح **Developer Tools** (F12) وتحقق من الأخطاء في Console
3. تأكد من أن أسماء الأعمدة صحيحة تماماً

### البيانات لا تتحدث بعد الرفع:
1. تأكد من أن GitHub قد استقبل التعديلات: `git push`
2. انتظر 30 ثانية حتى ينتهي Netlify/Vercel من إعادة النشر
3. امسح ذاكرة التخزين المؤقت (Cache) في المتصفح: `Ctrl+Shift+Delete`
